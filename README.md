# Youtube Clone

## Introduction
Cloned some of the essential features of Instagram
  * Sign in/up
  * Post feed
  * View feed
  * Follow/Unfollow user
  * Like/Unlike Post
  * Post comment
  * Chat with friends
  * Edit profile (change username, password, upload new image)
